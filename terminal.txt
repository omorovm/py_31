ctrl+alt+t  - открытие терминала
ctrl+shift+q - закрытие терминала (команда  exit)
директория = папка

/ -  эта корневая папка(здесь находятся все конфигурации по ОС)
~ -  домашняя папка

ls - показывает все папки в текущей директории
ls {название папки} - показывает все файлы и папке в написанной директории
ls -a - показывает все скрытые папки и файлы в текущей директории (-a  называется флагом)
ls -l - показывает более подробную информацию о папках и файлах в текущей директории


cd {название папки} - переходит в другую директорию
cd -  переходит в домашнюю папку
cd ~ - переходит в домашнюю папку
cd .. - возвращается на папку выше (back)
cd . - переходит в текущую папку (остаемся в ней же)

mkdir {название папки} - создать папку

rmdir {название папки} - удаление пустой папки
rm -rf - удаляет папку или файл со всем содержимым (безвозвратно)
rm -r  - удалит папку или файл (рекурсивное удаление)

mv {название файла} {путь куда будет перенесено/}- перезаписывает файлы в указанную директорию
mv {test test2} - переименует test на test2
touch {название папки} - создает файл с указанным расширением

sudo - дает права суперюзера (superuser do)

nano - открывает редактор кода
cat - показывает содержимое файла в терминале

pwd - показывает абсолютный путь (с начала корневой папки /) до текущей директории

code - открывает vscode
code . - открывает vscode в текущей директории (вместо . можно указать необходимую директорию)

!! - показывает последнюю введенную команду

history - показывает все последние команды введенные в терминала

echo 'hello world' > test.txt - вставка текста в файл

find - поиск файла в текущей дериктории
locate - поиск файла везде
